📦 Zařízení · Hardware

Co tu fyzicky stojí

Hub, sensorová síť přes 4 protokoly, 73 zařízení v 12 zónách. Postupně vrstvené hardware vrstvy, každá s vlastní rolí.

Legacy · 2016—2019

Homey Pro (Early 2019)

  • Single-core procesor, omezený výkon
  • Z-Wave + Zigbee + 433 MHz + Wi-Fi + IR
  • Flow-based runtime, omezená HomeyScript podpora
  • Výrobcem podporováno do EOL

Proč přestal stačit: S rostoucím počtem flows a prvními HomeyScripty hub začal mít problémy s výkonem. Paralelní skripty padaly na SIGABRT, runtime nestíhal real-time eventy, větší automatizace blokovaly UI. Pro jakýkoli script-first přístup prostě nebyl dimenzovaný.

Homey Pro 2026

Hardware specs

  • CPU: 1,5 GHz Quad-core ARMv8
  • RAM: 4 GB LPDDR4
  • Storage: 8 GB eMMC Flash
  • Updates garantovány do června 2031

Bezdrátové protokoly

  • Wi-Fi 2.4 GHz + 5 GHz
  • Bluetooth Low Energy 5.0
  • Matter v1.3 (controller)
  • Thread (Border Router capability)
  • Zigbee 3.0 (coordinator)
  • Z-Wave 700 Series
  • 433 MHz (legacy senzory)
  • Infrared (TV remote learning)
  • Volitelný Ethernet adapter (USB-C)

Co umí

  • Local-first processing — cloud jen jako backup, automatizace běží lokálně
  • Advanced Flow automation (zdarma, bez subscription)
  • HomeyScript runtime — full JavaScript pro custom skripty
  • REST API s Bearer tokenem pro externí integrace
  • Satellite Mode — extends range přes Homey Bridge
  • Experimental features: SSH access, virtual devices, power user tools

Co to umožnilo: 110+ skriptů paralelně bez SIGABRT, centralizovaná logika v JavaScriptu, request-first pipeline, daily Gemini analýza logů, real-time dashboardy přes REST API. Bez tohoto skoku by současný systém nebyl možný.

Bezdrátové protokoly

🔵

Zigbee

Páteř senzorové sítě. Sonoff SNZB-06P (mmWave presence), SNZB-03/04 motion + door/window, Aqara P1, IKEA Fyrtur roleta, Aqara Cube T1 Pro (6-stranný kontroler).

🟣

Z-Wave

Fibaro FGMS multisenzor (pohyb + lux + teplota) v kuchyni, FGS-214 relay pro topný okruh. Pomalejší ale stabilnější pro klíčové instalace.

📶

Wi-Fi

Xiaomi (Giuseppe robot vysavač, čistička vzduchu, senzory), Shelly Plug S (trouba/mikrovlnka/lednice — power monitoring), Nanoleaf Matter žárovky kuchyně.

🔊

Google Cast

Tři reproduktory: Nest Mini Kuchyn (primární výstup), Nest Hub Max Ložnice 2, Nest Mini Koupelna. Plus TTS pipeline s rate limiting na 200 char.

Klíčová zařízení

Roleta Fyrtur1 (jídelna) — privacy interlock pro open space.
Giuseppe (Xiaomi robot vysavač) — autonomní úklid s Homey scheduling.
Air Purifier 4 Lite — PM2.5 monitoring + auto mode + alerts.
Android TV ložnice — plná integrace přes com.android.tv:remote (31+ capabilities, native remote).
Aqara Cube T1 Pro — 6-stranný controller pro audio scény, světla, TV remote.
Raspberry Pi (WaveShare 10.1") — kiosk dashboard 1024×600 + dashboard proxy server na localhost:8080.
