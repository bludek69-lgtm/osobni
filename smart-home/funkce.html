🤖 Giuseppe

Vysavač jako autonomní agent

Xiaomi Viomi V7 robot vysavač, doma jménem Giuseppe (každé zařízení má jméno, je to přátelštější než „vysavač_01"). Wi-Fi, fixní IP, integrovaný v Homey přes Mi-Home komunitní app (driver vacuum_viomi_vacuum_v7 ověřeno v Homey API).

Plán úklidu

  • Schedule: sh_vacuum_router_v1 spouští úklid v okně, kdy nikdo není doma (geofence + telefon + scheduler).
  • Žádné rušení: pokud je v bytě motion (skutečný — ne falešný), úklid se odloží. Žádný robot vám neproletí kolem nohou.
  • Sleep guard: nikdy se neuklízí v noci ani když svítí sh_spim=yes.
  • Per-zone úklid: speciální flow „vyluxuj jen kuchyni" (po vaření, manuální trigger z dashboardu nebo z kostky).

Token security saga

Pairing Viomi v Homey vyžaduje cloud token z Mi Home. Token je session-only — rotuje při každém Wi-Fi reconnect, což znamená že po reset routeru / výměně Wi-Fi sítě se musí znovu získat. 5-hodinová epická saga: factory reset → AP mód (vlastní SSID viomi-vacuum-v8) → discover token → configure_wifi → re-pair v Homey. Po té už running stable.

Diagnostika

Battery, mode, last_clean_time, error_code se logují každou hodinu do EventLog. sh_self_healing_v4 má vlastní detector: pokud je Giuseppe stuck (battery se nemění + mode = error) → alert. Reálný incident: zachycená nožička stoličky, robot v dead-loop. Health detect to chytil dřív než já.
