Smart Home Dashboard

Co dashboard ukazuje

📊 Live data

Aktuální stav celého systému — topení, světla, audio, přítomnost, stav zón, senzory, rolety. Načítá data přímo z Homey přes API.

🧠 AI Brain Guardian

Meta-vrstva nad systémem — state validator, expected state engine, self-healing detekce. Health score a otevřené problémy v reálném čase.

🎛 Ovládání

Scény, kill switch, audio stop, override přítomnosti. Runtime akce — zap/vyp, scéna, nastavení režimu.

⚙️ Config Center

Trvalé nastavení systému — thresholds, intervaly, lux hodnoty, AI toggles. Editovatelné z dashboardu přes sh_cfg_* proměnné.

Sbírka App

Co sbírka umí

🪙 Katalog sbírky

Mince, bankovky, filatelie, pohlednice. Filtry podle kategorie, země, roku, stavu, AI confidence. Fulltext přes ID, motiv, text na kusu.

🤖 AI rozpoznávání

Fotka → AI návrh identifikace (typ, země, rok, nominál, motiv, stav, varianta). Každý návrh čeká na review — uživatel potvrdí, opraví nebo zamítne.

💰 Oceňování

Orientační odhad low/mid/high se zdrojem a datem. Varování na staré ceny a ceny bez ověřeného zdroje.

📤 Exporty

CSV, Excel (.xlsx), PDF seznam, ZIP archiv s fotkami. Reálná data nikdy na GitHub — pouze lokálně.

Pod kapotou

Architektura

Stack

  • Frontend: HTML/JS (PapaParse pro DEMO CSV)
  • Backend: Python + FastAPI (PRIVATE MODE)
  • Databáze: SQLite — lokálně, nikdy na GitHub
  • AI: Vision API (OpenAI GPT-4o / Anthropic)
  • Export: openpyxl, fpdf2, zipfile

Bezpečnost

  • Reálná data nikdy mimo lokální PC
  • GitHub: pouze zdrojový kód + demo data
  • PRIVATE instance: Cloudflare Tunnel + Access
  • Auth: Google login + TOTP (Authenticator app)
  • Pre-commit scanner — blokuje privátní data
