Stack

  • Homey Pro 2026
  • HomeyScript (JavaScript runtime)
  • Lokální SSD logging (JSONL + SQLite) na Raspberry Pi (primární zdroj eventů od 27.5.2026); Google Sheets jen archivní historická vrstva, už ne aktivní pipeline
  • Home Assistant OS (mirror + observer)
  • Gemini API (denní diagnostika)
  • HTML/CSS/JS dashboardy
  • GitHub jako single source of truth
