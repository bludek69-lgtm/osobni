📖 Pilíř · Příběh

Kdo stojí za SMART HOME OS

Není to projekt jednoho člověka. Není to ani projekt jediné AI. Je to kolaborace mezi člověkem a několika AI vrstvami — každá hraje jasně ohraničenou roli a každá přidává něco, co by sama nezvládla.

Human + AI Co-Built System

Role v projektu

5 vrstev co řídí systém

Architektura prvního principu. Každá vrstva má jasnou specializaci, žádné překryvy, jasný handoff mezi sousedními vrstvami.

👨‍💻

Hlavní architekt · Reality testing

Luděk Budínský

Návrh logiky domu, prioritizace toho, co dům má dělat. Reálný provozní testing každý den — žije v systému, který staví. Filozofie a UX rozhodnutí.

  • Návrh logiky a stavů
  • Use-case specifikace
  • Provozní reality testing
  • Prioritizace automatizací
  • Acceptance + iterace
🧠

Architectural Co-Pilot

ChatGPT

Vrstva která přemýšlí o systému z výšky. Audity, návrh architektury, „co bychom měli postavit dál". Brain Guardian koncepty a roadmap dokumentace.

  • Systémové audity
  • Architektonické návrhy
  • Optimalizační nápady
  • Brain Guardian koncepty
  • Roadmap + dokumentace
⚙️

Engineering Layer

Claude Code

Implementace kódu — script-first refaktoring, dashboard development, deploy přes Homey API, atomic patches s rollback. Vrstva která dělá, co architekt navrhne.

  • HomeyScript implementace
  • Dashboard development (V7)
  • API deploy + rollback
  • Refaktoring + LIB migrace
  • Engineering execution
✨

Embedded AI Layer

Gemini

Diagnostika a runtime briefing — vrstva která žije v systému 24/7 a sleduje provoz. Embedded AI co dělá ranní hlášení, navrhuje akce, kontroluje stav.

  • Provozní diagnostika
  • Ranní briefing (TTS)
  • Embedded AI vrstva
  • Anomaly detection asistence
  • 24/7 dohled
🛠

Coding Tooling Support

Cody

Podpůrný coding workflow, code review asistence, drobné refaktoring úkoly. Vrstva která zrychluje práci na detailech.

  • Code workflow support
  • Refactoring helpers
  • Pair-programming asistence
  • Code review pomocná vrstva

Architektura kolaborace

Pipeline myšlenky → kódu → systému

Každá role má jasný handoff k té vedlejší. Idea putuje shora dolů, runtime feedback putuje zpět nahoru.

👨‍💻
Luděk · Architect Definuje use-case, prioritu, acceptance criteria. Reálná zkušenost se systémem.
🧠
ChatGPT · Architectural Co-Pilot Audit současného stavu, návrh architektury, identifikace co lze udělat lépe.
⚙️
Claude Code + Cody · Engineering Píší kód, deployují přes API, testují, dokumentují. Konkrétní implementace návrhu.
🏠
Homey Pro 2026 · Runtime 79 zařízení, 100+ skriptů, 200+ flow. Fyzická realita — světla, topení, audio, senzory.
✨
Gemini · Embedded AI Layer Pozoruje provoz, diagnostikuje, briefuje. Vrstva mezi systémem a uživatelem.

Princip

Proč to funguje

Žádná z těch AI by sama systém nepostavila. ChatGPT umí plánovat, ale nezdeployuje skript. Claude Code umí implementovat, ale potřebuje jasné zadání. Gemini umí diagnostikovat, ale nedokáže navrhnout architekturu. Lidský architekt drží pohromadě intent, prioritu, kontext. AI vrstvy mu zrychlují každý krok 10×.
