🥗Meal Planner

Osobní plánovač jídelníčku — sestaví týdenní plán krabiček podle kalorického cíle, spočítá makra, nákupní seznam i co aktuálně mám v lednici. Fuzzy autocomplete, sledování expirace, recepty které can uvařit z toho co mám. Bez účtu, bez cloudu, vše lokálně.

React 18 TypeScript Vite 5 Tailwind CSS Tauri 2 Windows (.exe) PWA

Co to dělá

Proč vznikl: Zkoušel jsem MyFitnessPal, Yazio, Lifesum — všechny chtěly účet, posílaly reklamy nebo neuměly česká jídla. Tahle aplikace běží jen lokálně v prohlížeči nebo jako Windows .exe — profil, váha i jídelníček žijí v localStorage, žádný server nevidí co jím ani kolik vážím.

Jak funguje: Zadám výšku, váhu, věk, aktivitu. Aplikace spočítá denní kalorický cíl (vzorec Mifflin-St Jeor + activity multiplier). Při hubnutí navrhne asymetrický deficit, při nabírání svalů vybírá recepty s vyšším protein ratio.

Týdenní plán: Projde databázi receptů a sestaví 7 dní × 5 jídel (snídaně, svačina, oběd, svačina, večeře) tak, aby denní kcal padly do ±10 % cíle. Generuje nákupní seznam — sečte ingredience za celý týden, rozdělí do kategorií. Export CSV nebo tisk.

Lednice: Evidence zásob doma. Fuzzy autocomplete s Levenshtein algoritmem (překlepy ≤2 znaky). Sledování expirace s barevnými varovánímí (≤3 dny). Dashboard automaticky ukáže co brzy vyprší a jaké recepty lze uvařit z toho co je v lednici (≥50 % pokrytí ingrediencí).

Windows desktop: Tauri 2 instalátor (~1,1 MB NSIS .exe). Funguje jako normální desktopová aplikace bez Edgeu, bez app storu, bez aktualizací přes cloud.

Klíčové funkce

  • Auto-generovaný týdenní plán — 7×5 jídel, denní kcal v ±10 % cíle, penalizace opakování pro pestrost jídelníčku.
  • BMR/TDEE kalkulačka — Mifflin-St Jeor + activity multiplier. Asymetrický deficit/surplus podle cíle.
  • Doporučený cíl + banner — Když se posune doporučený kcal cíl, nabídne „Použít doporučený" — nikdy tichá změna.
  • Tracking váhy — Vlastní SVG sparkline graf, historie v localStorage. Žádné zdravotnické claims.
  • Optimizer pro hubnutí/nabírání — Vybírá recepty s vyšším protein ratio, těsnější dodržení kcal targetu.
  • Lednice s fuzzy matchem — Evidence zásob, Levenshtein autocomplete (překlepy ≤2), sledování expirace, varování při nízkém množství.
  • Dashboard widgety (Phase 8.1) — „V lednici brzy expiruje" (top 3, ≤5 dní) + „Co mohu uvařit" (top 3 recepty s ≥50 % pokrytím surovin).
  • Nákupní seznam s exportem — Agreguje z celého týdne, automaticky odečte zásoby z lednice. Export CSV nebo tisk.
  • Analytika — Kalorická odchylka za N dní, log váhy s grafem, přehled výdajů na jídlo.
  • Windows .exe + PWA — Tauri 2 instalátor ~1,1 MB. Funguje i offline jako PWA v prohlížeči.
